The goal is to automate the flow of supply chain data across multiple sources and generate meaningful KPIs for monitoring **inventory performance, procurement efficiency, supplier reliability, and logistics operations**.

This project includes:

- Automated data ingestion using **N8N**
- Data cleaning & transformation using **Quadratic**
- Data modeling using **SQL**
- KPI creation for critical supply chain metrics
- Automated alerts and workflow triggers  
- AI-assisted transformation and validation
